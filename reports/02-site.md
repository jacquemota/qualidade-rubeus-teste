BR-001 — [Hero Slider] Inconsistência de alinhamento entre títulos dos slides

Tipo: Melhoria
Classificação: Usabilidade
Prioridade: Baixa

Descrição:
Os títulos principais dos slides apresentam posicionamento horizontal diferente entre si.

No primeiro slide ("Graduação") o título inicia mais à esquerda, enquanto nos outros slides o alinhamento segue um posicionamento diferente dentro do layout.

Essa inconsistência quebra a continuidade visual do slider.

Passos para reproduzir

Acessar a página inicial do site

Navegar entre os slides utilizando as setas ou indicadores inferiores

Comparar o posicionamento do título principal

Resultado atual

Os títulos apresentam alinhamentos diferentes entre os slides.

Resultado esperado

Os títulos devem seguir um mesmo grid de alinhamento, mantendo consistência visual entre os slides.

BR-002 — [Hero Slider] Elementos decorativos inconsistentes entre slides

Tipo: Melhoria
Classificação: Desejabilidade
Prioridade: Baixa

Descrição

Os elementos gráficos decorativos (pontos e padrões geométricos) mudam de posição entre os slides sem manter uma lógica visual consistente.

Passos para reproduzir

Acessar a página inicial

Alternar entre os slides do banner principal

Resultado atual

Os padrões decorativos aparecem em posições diferentes sem padrão visual consistente.

Resultado esperado

Manter consistência no posicionamento dos elementos decorativos ou removê-los para simplificar o layout.

BR-003 — [Hero Slider] CTA inconsistente entre slides

Tipo: Melhoria
Classificação: Usabilidade
Prioridade: Média

Descrição

Os slides apresentam chamadas para ação (CTA) diferentes, apesar de pertencerem à mesma seção de destaque da página.

Exemplo:

Slide 1 e 2
→ botão INSCREVA-SE

Slide 3
→ botão SAIBA MAIS

Essa inconsistência pode gerar dúvida sobre qual ação principal o usuário deve executar.

Passos para reproduzir

Navegar entre os slides do banner principal

Observar os botões de ação

Resultado atual

Cada slide apresenta CTA diferente.

Resultado esperado

Manter consistência na ação principal da seção.

Exemplo recomendado:

todos com INSCREVA-SE
ou

todos com SAIBA MAIS
