# Processo Seletivo Qualidade Rubeus — Relatório de Itens

Este repositório contém a documentação dos itens identificados no teste prático de qualidade, incluindo:
- Relatórios por página (Certificação e Site)
- Evidências (prints, vídeos, logs)
- Checklists de validação
- Templates de reporte (bug/melhoria)

## Escopo
Páginas avaliadas:
1. Certificação: https://qualidade.apprbs.com.br/certificacao
2. Site: https://qualidade.apprbs.com.br/site

## Como navegar
- Relatórios:
  - `reports/01-certificacao.md`
  - `reports/02-site.md`
- Evidências:
  - `evidencias/certificacao/`
  - `evidencias/site/`
- Checklists:
  - `checklists/checklist-manual.md`
  - `checklists/checklist-ux-ui.md`

## Convenções
- IDs no formato: `BR-###` (ex: BR-001)
- Nome de evidências deve conter o ID:
  - `evidencias/certificacao/BR-001.png`
  - `evidencias/site/BR-010.mp4`
